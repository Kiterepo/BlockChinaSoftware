Anti-China-Anit-virus
=====================

国产安全软件免疫补丁

Introduction / 简介
=====================
国产安全软件免疫补丁是利用CRL，将这些开发商的证书导入吊销列表中，在运行含有这些证书的程序时会自动被系统阻止的原理制作而成的。

Installation / 安装
=====================
点击下面的URL下载，按照提示导入注册表即可。

Dependencies / 依赖
====================
Windows 7或更新版本的UAC组件
